# Renting a sewing machine from the library

不只是书——从图书馆借一台缝纫机如何能促进民主

芬兰的图书馆正越来越多地不以借出多少书来衡量价值，而是以它们如何帮助社会运转来评判。

在赫尔辛基一个寒冷的一月清晨，大约20人聚集在城市中央图书馆Oodi门外，等待开门。

"当我看到人们几乎在早上八点跑进大楼，直奔他们最喜欢的位置时，我眼中含着泪水，"芬兰首都图书馆服务总监卡特里·万蒂宁说。"这说明图书馆真正属于公众。"

到午餐时间，大楼已经挤满了人，访客们在楼层间穿行寻找空座位。学生们在俯瞰芬兰议会大厦的巨大窗户旁用笔记本电脑工作，父母们在色彩明亮的游戏区与婴幼儿一起阅读。

一小群人围坐成一圈：他们在织毛线袜，有经验的人帮助新手掌握技巧和花样。在图书馆的音乐舱里，一位中年男子正在录制他的第一首萨克斯曲目。在图书馆咖啡厅，一位老年女士为两个外国女孩举办芬兰语会话课。在入口处，一个十几岁的男孩拿起他借的篮球，加入朋友们在图书馆外的球场上打球。

Oodi在2019年被评为世界最佳新建图书馆，正蓬勃发展——这与许多其他国家公共图书馆正在慢慢消失的状况形成鲜明对比。

2008年至2019年间，美国关闭了766家公共图书馆，而在英国，2016年至2023年间，超过180家由市政运营的图书馆被关闭或移交给志愿者团体。芬兰则相反，正在扩展图书馆并将其转型为公共资助的社区服务中心，人们可以借到的远不止书籍。

来自这些举措的研究——不仅在芬兰，还包括挪威、瑞典、丹麦和加拿大——已经表明图书馆在促进社会包容方面发挥着重要作用，提出了一个发人深省的论点：如果图书馆的价值不在于借出多少书，而更在于如何帮助社会运转呢？世界能从芬兰模式中学到什么？

**不只是书**

芬兰拥有超过700家图书馆，服务560万人口，提供从播客录音室和3D打印到网球拍和游泳池通行证等各种服务。

据万蒂宁介绍，赫尔辛基图书馆中书籍之后借用最多的是空间：可以免费预订的房间，用于会面、学习、举行政治讨论或制作音乐。在可携带物品中，桌游和主机游戏排名最高。

万蒂宁解释说，这种借用文化植根于深厚的务实主义，可以追溯到芬兰的农村历史，当时人们经常共享农业机械。"如今，城市中许多人住在小房子里，他们可能一年只需要一次缝纫机，"万蒂宁说。"那为什么要买一台呢？人们更愿意不花自己的钱，因为他们可以免费使用通过税收资助的缝纫机。"

在赫尔辛基以北600公里处，奥卢市新翻修的中央图书馆Saari体现了同样的理念，图书馆职员克里斯·斯蒂芬森一边为一位访客加载微缩胶片阅读器以浏览旧报纸一边说道。

在他周围，读者们坐满了柔和灯光下的长桌。一位刚退休的教师正在为他参加的合唱团和吉他乐队打印乐谱。楼上，一个年轻人在预约了缝纫机时段后来修短牛仔裤。同一房间里，一台3D打印机在一个用热压机为朋友生日制作自己设计的T恤的女学生身后嗡嗡作响。一台激光切割机闲置一旁。

根据政府报告，55%的芬兰人每月至少访问图书馆一次。文化和教育部的数据显示，芬兰人每年使用图书馆9.1次。在英国，数据分析表明一个人平均每年大约访问2.5次；在美国为2.4次，欧盟平均约为3.5次。

斯蒂芬森在移居芬兰之前，在英国图书馆工作了20年。"我看到许多图书馆被关闭，社区失去了重要的东西，"他说。

据奥卢大学信息研究教授诺拉·希尔沃宁称，削减利用率不高的服务以节省资金可能会成为一个自我实现的预言。"我们先削减图书馆的开放时间，结果图书馆访问量下降，"希尔沃宁说。"然后这又被用作进一步削减甚至关闭图书馆的理由。"

相反，希尔沃宁认为，机构应该问为什么服务没有被使用。是因为人们觉得它没有价值，还是他们无法获得它，或者他们不知道它的存在？

"使用率不仅反映了服务的价值：它还受到可见性和可获得性等因素的影响，"希尔沃宁说。

**民主的支柱**

但芬兰图书馆的意义远不止工具和会议室。

教授、失业者和无家可归者都使用同样的图书馆空间，使图书馆成为芬兰民主基础设施的关键组成部分，希尔沃宁说。"它们是任何人都可以获取知识、结识他人和参与公共辩论的场所，不论收入或背景，"希尔沃宁说。

虽然这些是世界各地图书馆学的核心价值观，但希尔沃宁说，在北欧国家，它们被写入了法律。根据芬兰《图书馆法》，公共图书馆必须促进民主、言论自由和积极的公民参与。（一些其他北欧国家也有类似的政策。）

芬兰的投入反映了这一承诺：2025年，该国在公共图书馆上花费了近3.71亿欧元（4.3亿美元/3.21亿英镑）——人均65.78欧元（76美元/57英镑），相比之下，英国人均花费约10英镑（13.5美元），美国公共图书馆总支出为152亿美元（114亿英镑），即人均45美元（34英镑）。

"图书馆可以直接支持民主——这是世界可以从芬兰学到的一件事，"德克萨斯大学图书馆学者R·大卫·兰克斯说，他认为当社区积极利用图书馆来学习、辩论和共同创造知识时，图书馆就会蓬勃发展。

例如，作为芬兰国家数字支持模式的合作伙伴，许多芬兰图书馆提供广泛的数字支持，使来自各种背景的公民能够获得国家的数字福利服务。

图书馆员帮助用户处理在线事务，从税务服务和银行账户到养老金门户和数字健康记录，他们还经常协助撰写简历和求职申请。因此，最近一项关于芬兰图书馆的研究得出结论，图书馆作为关键的包容性基础设施发挥着作用。

"这促进了包容性，"奥卢地区图书馆服务经理梅尔维·瓦拉说。"图书馆就像每个人的共享客厅。"

因此，一项对全球38项研究的分析发现，公共图书馆持续产生超过其成本的价值，每投入一美元回报三到五美元。该研究强调，图书馆创造了直接效益——如借用而非购买带来的节省以及对求职者的支持——以及间接效益，包括提高识字率、数字能力、就业能力和社区福祉。

在芬兰地方当局进行的年度评估中，图书馆始终位居最受重视的公共服务榜首。"我们几乎覆盖了所有人，不论社会地位或文化背景，"万蒂宁说。"这就是真正的日常民主。"
